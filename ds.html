Chapter 1: Array

Array is a collection of elements stored in contiguous memory.

Explanation: Array allows indexed access, fixed size, and is used for fast read/write operations by index. Pune University topics include 1D, 2D arrays, operations (traverse, insert, delete, search).

Chapter 2: Linked List

Linked list is a collection of nodes connected using pointers.

Explanation: Linked list supports dynamic size and easy insertion/deletion without moving existing elements; includes singly, doubly, circular lists. In syllabus, you should cover insertion, deletion, searching and traversal complexity.

Chapter 3: Stack

Stack follows LIFO (Last In First Out).

Explanation: Stack is used in recursion management, expression evaluation, and backtracking. Learn push, pop operations, and stack overflow/underflow handling (semaphores, memory stack).

Chapter 4: Queue

Queue follows FIFO (First In First Out).

Explanation: Queue is used in process scheduling, buffers, breadth-first search. Study linear queue, circular queue, priority queue and dequeue with insertion/ deletion cases.
