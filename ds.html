Chapter 1: Arrays

Definition: Array is a collection of elements stored in contiguous memory locations.

Key Concepts: Fixed size, random access O(1), insertion/deletion O(n)

Pune University Focus: 1D and 2D arrays, array operations, memory representation

Practice Exercises:

  • Find maximum element in array
  • Rotate array by k positions
  • Find missing number in 1 to n
  • Two sum problem (LeetCode)

Chapter 2: Linked Lists

Types: Singly, Doubly, Circular Linked Lists

Advantages: Dynamic size, easy insertion/deletion

Operations: Insert, delete, traverse, reverse, detect cycle

Practice Exercises:

  • Reverse a linked list
  • Detect and remove cycle
  • Find middle element
  • Merge two sorted lists

Chapter 3: Stacks & Queues

Stack (LIFO): Push, pop, peek operations

Queue (FIFO): Enqueue, dequeue operations

Applications: Expression evaluation, function calls, scheduling

Practice Exercises:

  • Valid parentheses problem
  • Implement stack using queues
  • Circular queue implementation
  • Next greater element

Chapter 4: Trees

Types: Binary Tree, BST, AVL, B-Tree

Traversals: Inorder, Preorder, Postorder, Level order

Operations: Insert, delete, search, height, diameter

Practice Exercises:

  • Binary tree traversals
  • Check if BST
  • Lowest common ancestor
  • Serialize/deserialize tree

Chapter 5: Graphs

Representations: Adjacency matrix, adjacency list

Algorithms: DFS, BFS, Dijkstra, Kruskal, Prim

Applications: Shortest path, minimum spanning tree, topological sort

Practice Exercises:

  • Graph traversal (DFS/BFS)
  • Detect cycle in graph
  • Topological sorting
  • Find shortest path

Chapter 6: Sorting & Searching

Sorting: Bubble, Selection, Insertion, Merge, Quick, Heap sort

Searching: Linear, Binary, Hashing

Complexity: Time and space analysis

Practice Exercises:

  • Implement all sorting algorithms
  • Binary search variations
  • Find kth largest element
  • Sort array with 0s, 1s, 2s
