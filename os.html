🖥️ Operating System

Operating System Syllabus (Pune University)

  • Concurrency: processes, threads, synchronization
  • Scheduling algorithms: FCFS, SJF, Round Robin, Priority
  • Memory management: paging, segmentation, virtual memory
  • File systems, I/O systems, device management
  • Deadlock: detection, prevention, avoidance, recovery
  • Modern OS concepts: virtualization, cloud, containers

Study Material Resources

  • Textbook: "Operating System Concepts" by Silberschatz
  • Textbook: "Modern Operating Systems" by Tanenbaum
  • C programming lab exercises and OS lab notes
  • Online: GeeksforGeeks OS tutorials, practice OS problems

Key Practical Skills

  • Write code for and simulate scheduling algorithms
  • Solve memory allocation problems (first-fit, best-fit, worst-fit)
  • Implement producer-consumer, readers-writers problems
  • Study Linux commands and process monitoring tools

Topic-wise Explanation

Process Synchronization

Understand critical section, mutex, semaphore, monitors, and algorithms like Peterson's. Practice solving classical synchronization problems.

Memory Management

Learn paging, segmentation, TLB, page replacement algorithms (FIFO, LRU, Optimal). Pune University exam often asks to compute page faults.
