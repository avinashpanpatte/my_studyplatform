🛠️ Software Engineering

Software Engineering Syllabus (Pune University)

  • Software development lifecycle (SDLC) models: Waterfall, Agile, Spiral, V-model
  • Requirements engineering and specification
  • Design: architectural patterns, UML diagrams, design principles
  • Software testing: unit, integration, system, acceptance, test cases
  • Software quality assurance and maintenance
  • Project management, risk management, configuration management

Study Material Resources

  • Textbook: "Software Engineering" by Ian Sommerville
  • Textbook: "Fundamentals of Software Engineering" by Ghezzi
  • Pune University SE notes and case studies
  • Online: Guru99 Software Engineering tutorial
  • Practice: Create UML models with diagrams.net, test plans from case problems

Key Practical Skills

  • Make SRS (Software Requirement Specification) documents
  • Create use-case diagrams, class diagrams, sequence diagrams
  • Implement project diagrams in team assignments
  • Write structured test cases and traceability matrix

SDLC Models

Understand advantages/disadvantages of Waterfall vs Agile vs Spiral; make a comparison table for exam notes.
