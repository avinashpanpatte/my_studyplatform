🗄️ DBMS (Database Management System)

DBMS Syllabus (Pune University)

  • Introduction to DBMS: data models, schemas, instances
  • ER modeling, relational model, constraints
  • Relational algebra, SQL (DDL, DML, DCL)
  • Normalization (1NF, 2NF, 3NF, BCNF)
  • Transaction management, concurrency control and recovery
  • Indexing, file organization, query optimization

Study Material Resources

  • Textbook: "Database System Concepts" by Korth & Silberschatz
  • Textbook: "Fundamentals of Database Systems" by Elmasri & Navathe
  • Pune University DBMS lecture notes and previous year papers
  • Online: TutorialsPoint DBMS, GeeksforGeeks SQL and DBMS guides
  • Practice: SQLZoo, LeetCode Database problems, HackerRank SQL tasks

Key Practical Skills

  • Design ER diagrams from case studies
  • Convert ER to Relational schema
  • Write complex SQL queries with joins and nested queries
  • Understand ACID, 2PL, deadlock handling
  • Implement simple indexing and query plan analysis

Topic-wise Explanation

Relational Algebra

Relational algebra is a procedural query language using operators like select, project, join, union, intersect and difference. Pune University emphasizes practice problems on joining student and course tables.

Normalization

Normalization avoids redundancy; understanding 1NF, 2NF, 3NF, BCNF with examples is critical for exams. Practice decomposition of table schemas into normalized forms.
