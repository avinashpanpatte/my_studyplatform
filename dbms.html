🗄️ DBMS (Database Management System)

DBMS Syllabus (Pune University)

  • Introduction to DBMS: data models, schemas, instances
  • ER modeling, relational model, constraints
  • Relational algebra, SQL (DDL, DML, DCL)
  • Normalization (1NF, 2NF, 3NF, BCNF)
  • Transaction management, concurrency control and recovery
  • Indexing, file organization, query optimization

📚 Recommended Textbooks

"Database System Concepts" by Abraham Silberschatz, Henry F. Korth, S. Sudarshan

Edition: 7th Edition (most recent)

Chapters for syllabus: 1-3, 6-8, 10-12, 14-16, 17-19

Key features: Comprehensive coverage, real-world examples, excellent for theory

"Fundamentals of Database Systems" by Ramez Elmasri, Shamkant B. Navathe

Edition: 7th Edition

Best for: ER modeling, normalization, SQL practical examples

Strength: Step-by-step explanations, case studies

"Database Management Systems" by Raghu Ramakrishnan, Johannes Gehrke

Focus: Query optimization, transaction management, advanced concepts

Level: Advanced undergraduate level

Key Practical Skills

  • Design ER diagrams from case studies
  • Convert ER to Relational schema
  • Write complex SQL queries with joins and nested queries
  • Understand ACID, 2PL, deadlock handling
  • Implement simple indexing and query plan analysis

📖 Detailed Chapter Breakdown

Chapter 1-2: Introduction & ER Model

Key Concepts: Data models, schemas, instances, ER diagrams, entities, attributes, relationships

Cardinality: 1:1, 1:N, M:N relationships with examples

Practice Exercises:

  • Draw ER diagram for library management system
  • Convert ER to relational schema
  • Identify weak entities and generalization

Chapter 3-4: Relational Model & SQL

Relational Algebra: Select, Project, Join, Union, Intersection, Difference

SQL Commands: DDL (CREATE, ALTER, DROP), DML (INSERT, UPDATE, DELETE), DQL (SELECT)

Practice Exercises:

  • Write SQL queries for complex joins
  • Create database schema for university system
  • Implement nested queries and subqueries
  • Practice aggregate functions and GROUP BY

Chapter 5-6: Normalization

Normal Forms: 1NF, 2NF, 3NF, BCNF, 4NF, 5NF

Functional Dependencies: Partial, transitive, multivalued dependencies

Practice Exercises:

  • Decompose tables to BCNF
  • Identify normalization violations
  • Lossless join decomposition
  • Dependency preservation

Chapter 7-8: Transaction Management

ACID Properties: Atomicity, Consistency, Isolation, Durability

Concurrency Control: 2PL, Timestamp ordering, Optimistic protocols

Recovery: Log-based recovery, ARIES algorithm

Practice Exercises:

  • Schedule serializability analysis
  • Deadlock detection and prevention
  • Log file analysis for recovery

Chapter 9-10: Indexing & Query Processing

Indexing: B-Tree, B+-Tree, Hash indexing

Query Optimization: Cost estimation, query plans, join algorithms

Practice Exercises:

  • Compare different indexing techniques
  • Analyze query execution plans
  • Optimize slow queries
